Description             :	Three levels which has BJ infiltrate a hidden Nazi
				outpost, underground prison cave and small palace which
				is currently holding an important meeting.

Installation		:	Backup the following files in your wolf3d/base folder;
					gamemaps.wl6
					maphead.wl6
				Then copy the zip contents into the same folder and run
				Wolf3d via either DOS or Wolf4SDL source port.

				This release may be repackaged in future to be played
				out the box and hassle free. Hopefully.
