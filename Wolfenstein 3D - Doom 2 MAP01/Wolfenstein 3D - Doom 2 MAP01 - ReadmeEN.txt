Wolfenstein 3D - Doom 2 MAP01
=============================

Info
====
Doom 2 MAP01 for Wolfenstein 3D full and demo versions

Install
=======
Copy the correct folder's content to the game's folder overwriting the original files.
- Full: wl6 folder
- Demo: wl1 folder

In the export folder you can find the map in multiple editor formats.
It was made with Havoc's Wolf3D Editor, it's recommended to import the map with it.

Map load
========
Start the first episode from the menu
