Game uses the same engine as Trail of Schabbs SDL-demo (2010).
For the same reason the in-game palette and some textures are taken from TOS.

As the original storyline had so many gaps, it was slightly altered...
Because of these changes, some bossfights were altered as well.

Depending on the difficulty, some enemy behavior changes.

5th difficulty setting was originally intended to start a completely different game. Instead of the regular levels, player was going to have to play through series of challenge levels.

Shotgun was originally planned to be a double-barreled one and the rifle was going to be equipped with a scope (zooming was already included in the games code).

Enemies reloading their weapons was not planned to be a part of the game.
It was later added to balance things out and make the game a bit easier.

As mutants were based on dogs, they had no standing frames. Standing frames were added as it was hard to design athmospheric levels with just the mutants patrolling the areas.

All bosses have re-recorder voicefiles.

There is a bug in the first level of the demo that causes all guards to become alerted when the level finishes loading.

Originally there was going to be a secret level "The forest camp", where player would fight tanks with panzershreks. It was removed as it was too hard and did not fit the games storyline.

As an early concept, some of the knight armors in Sturmgard would attack the player. These knights would act like dogs and use only melee attacks, and to make things more interesting - you could only harm them with STG.
