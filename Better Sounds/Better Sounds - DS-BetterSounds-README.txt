This mod contains 4 files:
- Digital sounds (replaces all Adlib sounds with digital ones from various sources)
- MIDI Music (replaces music with MIDI versions from the wolfmidi project)
  - MT32 (emulated Roland)
  - GUS (emulated GUS)
  - Windows (Windows default soundfont)
