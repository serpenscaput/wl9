             NOTE:  This is a modified version by Colonel Bill.

                        Description of modifications:

               Combined both of Bill Neisius' patches into one.
                   Added sounds. (The originals had none)
                        Added new maps (10 new floors)
                        Added some new wall textures

            There was sort of a strange atmosphere with the original
            patches (Pac-men and clowns with the nazi theme).
            I was originaly going to get rid of all the nazi stuff,
            then I quickly realized that it just wasn't the same. 
            So I decided to expand on the Hitler\nazi theme.

                               Here's the story:

            You find yourself transported to a strange alternate
            universe. This world is inhabited by strange creatures.
            They seem to worship Hitler. Your goal is to find the
            source of this bizzare phenomena. You will eventually
            find out that you were not the only one from Earth that
            was sent to this world.
            Your ultimate goal is to blast this other Earthling back
            to his hell hole. That is the only way this alternate
            universe will get back to normal, and also the only way
            you can get back home. (Of course, you can pick up all
            the treasure you can carry on the way)!

                                    ------

                           How to use this patch:

                       This patch consists of 3 files:
                   VSWAP.WL1, GAMEMAPS.WL1, and MAPHEAD.WL1
                  Copy these files to your Wolfenstein folder.
                 This will overwrite the original files with the
             same names, so you may want to back them up before hand.
                      Start the game just like normal.

      NOTE: This patch is made for Wolfenstein 3-D Version 1.4 (shareware)
