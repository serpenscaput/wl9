General

WDC is a Windows editor for ID Software's Wolfenstein 3D and Spear of Destiny games.
It also has limited functionality with the following games:
  • Blake Stone 1 and 2
  • Operation Body Count
  • Corridor 7 (All versions)
  • Catacomb 3D
  • Super Noah's Ark 3D
  • The Catacomb Trilogy
  • Commander Keen 4-6
See General :: Game Compatibility for a full list of functionality for these games.
