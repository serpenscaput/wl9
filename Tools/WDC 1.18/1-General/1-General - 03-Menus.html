File
New Project Start a new project.
Open Project Open an existing project.
Save Project As Save your current project under a different file.
View Data Files A quick way to simply look at some data files without the hassle of a project file.
Compile All Compiles all the data files for your project.
Compile Use the sub-menus to compile a single data file.
Rebuild VGADICT If WDC says some pictures are too big to fit in the VGAGRAPH file, try running this to make the VGAGRAPH encryption better fit your data files..
Project Information Opens a window that lets you change the base folder, compiled extension, output folder, and test program.
Test Project Runs the test program (If one has been set up for this program).
Code Editor Runs the code editor you have specified in File->Preferences.
Preferences The window has a setting called "Maintain file associations" which will tell WDC whether or not it should re-associated *.wdc files with it on program startup. Uncheck this if you have another program that uses *.wdc programs.
You can also setup which programs you want to use to edit BMP and WAV files. This is also where you setup the code editor. If the automatic conversion to 256 color BMPs messes you up, disable it here.
Most Recent List WDC also keeps track of the 4 most recently opened projects.

Edit
Undo Reverts back to the state prior to the previous change. Only available while editing maps.
Redo Re-applies a change that you undid. Only available while editing maps.
Cut Copies a map segment to memory and removes it from the map. Only available while editing maps using the "Select" tool.
Copy Copies a map segment to memory and leaves it in the map. Only available while editing maps using the "Select" tool.
Paste Applies a map segment from memory. Only available while editing maps using the "Select" tool.
Edit Data If available, an edit window will appear so you can edit the associated file. If there is no associated file, you will be asked to export the original data to a file before editing.
Verify Data Files Checks to see if all associated files exist. File association can be broken if you move or delete an associated file. You will be asked to fixed broken associations.
Data Statistics Brings up a window that displays how many of each type of data chunk has been replaced and how many have been added.
Import/Export
Insert File Associates a file with the current data chunk.
Export Chunk Export the original data chunk to an external file. The file type will be dependent upon the data being viewed.
Use Original Data Removes the file association for this data chunk.
Add Chunk Inserts a new data chunk to the file immediately before the current chunk.
Add Chunk to End Inserts a new data chunk at the very end of the file section.
Remove Chunk Removes an added chunk.
Import Multiple Files Use this to replace a bunch of chunks at once.
Insert Multiple Files Use this to add a bunch of chunks at once.
Export Multiple Chunks Allows you to select and export multiple chunks from AUDIOT, VGAGRAPH and VSWAP.
Import Map to Clipboard Imports a map into the clipboard so you can paste whenever you'd like.
Paste Map from Clipboard Opens a list of everything you've copied or cut so you can select what you want to paste.
Import Map Imports a map straight into your map file.
Export Multiple Maps Exports multiple maps into one file.
View
All Items Simply changes the data screen to whatever you select. These correspond to the graphical view buttons on the window.
Picture Tools
Save Image With Current Palette Save the image in the current palette using color-matching.
Save New Image as 8-Bit Converts the current 24-bit image to 8-bit using color-matchin and the current palette.
Convert to Hi-Res Converts all wall and sprite images to 128x128 images.
Convert to Current Palette Converts the selected group of pictures to the current palette using color-matching.
VGAGRAPH Chunks Lets you look at a listing of all the VGAGRAPH chunks.
Analyze Palette Gives a detailed listing of how much the colors are used, what images use the colors, and whether some colors are not used at all.
Sound Tools
Save WAV in Wolf3D Format Save non-Wolf3D format WAV files into Wolf3D's WAV format.
"Add a Sound" Wizard A wizard that makes adding a new sound easy.
Generate AUDIO*.H file Generates code based on the given sound chunk names for use in AUDIO*.H.
Generate wolfdigimap enum Generates code based on the given sound chunk names for use in WL_MAIN.C.
Sound Chunk Names Lets you change the names of the sound and music chunks.
Sound Priorities Allows you to easily alter the adlib and pc sound priorities.
Map Tools
Clear Map (Solid) Clears the map using the wall code selected for the left mouse button.
Clear Map (Random) Clears the map using a random mix of the wall codes selected for the left and rightmouse buttons.
Adjust Object / Door Direction
Rotate Use this to rotate your map clockwise by either 90, 180, or 270 degrees.
As of version 1.2.104, this also rotates guards, turning points, starting points, and doors.
As of version 1.7.195, you can rotate non-square maps.
Flip Use this to flip your map horizontally or vertically.
As of version 1.2.104, this also flips guards, turning points, and starting points.
Advanced Gives you commands that you can use at the plane-level.
Prefab Allows you to save and use already-made map segments.
Save Map as BMP Saves the current map as a 16-color BMP.
Map Tile Count Simply counts the individual wall and object types are displays the totals.
NOTE: The plane 3 list only appears if the map has something in plane 3.
Autofloor This option will automatically given rooms their own floor code. This will not overwrite the ambush/deaf guard or secret elevator floor codes. Secret walls are also taken into account when assigning floor codes.
Error check map Checks the current map for common map errors. You can choose to overlook these errors and WDC will compile your maps even if it says there are errors. For a full list of map errors that WDC current detects, check out the Error Message Guide :: Map Errors.
Clear Error Markers Clears all the error markers from the map view.
Map Difficulty This simply calculates a difficulty value for each difficulty level for the map. Difficulty is currently only based upon guard hitpoints. Health and ammo packs are not taken into account.
Generate Test Map Clears the current map and creates a new one that includes all walls and objects.
Map Options
Show {item} Toggles whether {item} map tiles are drawn in the map area.
Int (16-bit) Mode The default map editing mode. When in integer mode, map changes affect the entire map code. Numeric values can range from 0 to 65535.
High Byte Mode When in high byte mode, map changes only affect the high byte of the map code. Numeric values can range from 0 to 255.
Low Byte Mode When in low byte mode, map changes only affect the low byte of the map code. Numeric values can range from 0 to 255.
Map Symbols Opens up the map symbol editor. More information on this screen can be found at Editing :: Map Symbols.
Map Difficulty Adjustments Allows you to set up the values used to calculate the map difficulty.
Map Zoom You can select a different zoom level for the map. Level 7 is the default and uses a 7x7 icon. Each addition level makes the icon one pixel wider and taller.
Map Tile Tooltips Allows you to change text that appear when you hover over the top row in the map editor. This text can be used to remind you what a certain map tile does if you have given it a special meaning.
User Tools
Configure Opens a screen from which you can set up your own user tools. These tools can be accessed using CTRL+{numeric} or one of the menu items on this menu. Additional information can be found in General :: User Tools
Help
Contents Opens up this help file.
FAQ Takes you to the FAQ
Game Compatibility Takes you to General :: Game Compatibility
Tutorials Takes you to Tutorials
Homepage (Online) Takes you to WDC's homepage.
Forums (Online) Takes you to the WDC forum where you can post bugs, requests, or just about anything.
Update Checks the web for WDC updates.
About Lets you know what version of WDC you are using as well as some additional information.
