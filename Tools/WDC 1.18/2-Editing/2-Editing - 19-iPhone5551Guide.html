Editing :: iPhone .5551 Guide

Walls:
  • No bounds.
  • Version 1.1 always uses PVRTC_RGB_4.
  • Version 1.0 always uses R5G5B5X1.
Sprites:
  • Set bounds.
  • Always uses R5G5B5A1.
Pictures:
  • Version 1.1 mainly uses R5G5B5A1.
    background_1 uses PVRTC_RGB_4.
    intermission_256 and victory_256 use PVRTC_RGBA_4.
  • Version 1.0 always uses R5G5B5A1.
Notes:
  • All of the original files include mip maps.
  • The original files are never R4G4B4A4, PVRTC_RGB_2, or PVRTC_RGBA_2, but the engine supports these types.
