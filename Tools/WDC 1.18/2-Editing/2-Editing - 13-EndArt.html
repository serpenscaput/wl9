Wolfenstein 3-D/Spear of Destiny/Commander Keen Commands
Command Action
; Comments the line.
^> Move the text starting position to the center of the current line.
Not used. Untested.
^By,x,w,h Draw a box at x,y having width w and height h.
Not used. Untested.
^Chh Change the text to color 0xhh.
For Commander Keen, this command is ^Ch.
^E Marks the end of the data.
The rest of the line will be ignored.
^Gy,x,n Insert VGAGRAPH graphic #n at pixel x,y.
^Ly,x Move to pixel x,y.
^P Starts a new page.
The rest of the line will be ignored.
^Ty,x,n,t Insert VGAGRAPH graphic #n at pixel x,y after t tics.
Only used in Commander Keen games.
Note: The READTHIS chunk in Wolfenstein 3-D only appears correctly with v1.1 data files.
