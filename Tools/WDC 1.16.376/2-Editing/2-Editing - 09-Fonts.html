Editing :: Fonts

Fonts are pretty easy to change.


  1. Scroll the bar to select the character you want the change.
  2. Adjust the width of the character, if you want to.
  3. Draw while holding the left button to make a square white.
  4. Draw while holding the right button to make a square black.
The source code will determine the actual foreground and background colors used when the font is used.
