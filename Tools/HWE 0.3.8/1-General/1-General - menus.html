Menus and shortcuts

This section explains to you all menus and shortcuts which are offered to you by HWE.

File

Open Project Ctrl+O Open an existing project.
Save Project Ctrl+S Save the changes to your current project.
  Save Project As...   Save the changes to your current project under a different WAD filename. (This option is only enabled, when the "Save maps into WAD file" option is enabled.)
Import Map Shift+Ins Import map(s) from different Wolfenstein 3D editor formats.
Export Map Shift+Del Export map(s) in different Wolfenstein 3D editor formats.
  Exit Alt+F4 Exit HWE.

Edit

Undo Ctrl+Z Undo last operation. (Unlimited)
Redo Ctrl+Y Redo last operation. (Unlimited)
Cut Ctrl+X Cut the map selection and remove it from the map.
Copy Ctrl+C Copy the map selection and leave it in the map.
Paste Ctrl+V Paste the last copied object into the map.
Clipboard Manager Ctrl+B Open the clipboard manager (you can decide which map to paste into the map).
Find & Replace Ctrl+F Search or replace walls / objects in a map.

Map

Map View Options F2 This option let you decide which planes or objects to show on the maps.
Add Map Ins Insert a new map to the file immediately before the current map.
Delete Map Del Delete maps in the file immediately before the current map.
Autofloor Ctrl+Alt+F This option will automatically insert floor codes in all rooms of your map. This will not overwrite the ambush/deaf guard or secret elevator floor codes.
Previous Level Ctrl+P Load prior level / map.
Next Level Ctrl+N Load next level / map.
Go To Level Ctrl+G Jump to the desired level / map directly.
Plane Manager Ctrl+D Open the map plane manager (you can decide which features the map planes shall have).
  Map Symbols Ctrl+E Open the map symbols manager (you can decide which features the map symbols shall have).
  Map Properties Ctrl+M Open the map properties manager (you can decide which features the maps shall have).
  Map Information Ctrl+I This option shows you the amount of various map type objects on your current map.
  Map Configurations Ctrl+Alt+C This option let you decide which map compression, file format, and what compression tags shall be used, when saving the maps.
  Clear Map (Solid)   This feature erases the current map, and fills it with the wall tile selected for the left mouse button (or with the first wall tile, if the tile on the left mouse button isn't a wall tile).
  Clear Map (Random)   This feature erases the current map, and fills it randomly with the wall tile selected for the left mouse button (or with the first wall tile, if the tile on the left mouse button isn't a wall tile).
  Advanced   You find additional map tools in this menu, like flipping planes or rotating maps (or copied map parts).
  Display Map Grid Ctrl+W This option let you activate or deactivate a grid for the map editing area.
  Save Map As BMP Ctrl+Alt+S This option let you take a screenshot of the current map.
  Int (16-bit) Mode Ctrl+N Edit objects and planes in default map editing mode.
  High Byte Mode Ctrl+H Edit objects and planes in high byte mode only.
  Low Byte Mode Ctrl+L Edit objects and planes in low byte mode only.

? (Help)

  Help F1 Open this help file.
  Check For Update   Let you know if there's a new version of HWE available for download.
  Check For Update At Startup   This option let you decide if HWE shall automatically check for a program update at startup.
  About   Let you know what version of HWE you are using and how to contact me.

Keyboard Shortcuts

  F3   Find / replace next tile or object.
  Ctrl+PageUp   Go to the next level / map.
  Ctrl+PageDn   Go to the prior level / map.
