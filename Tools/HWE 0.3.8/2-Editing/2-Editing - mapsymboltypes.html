Map symbol types manager

This dialog is accessible via the map symbols manager.

Select map symbol type

You can change the order of the map symbol types by selecting the map symbol type you want to move and using the arrow buttons to the right of the list.

Edit map symbol type information

  • Name - The name for the map symbol type.
  • ID - The ID for the map symbol type.

Note: The ID must consist of a single character or several characters (written as one word) without blanks / spaces!
