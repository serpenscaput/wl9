Welcome to the first Team Fir2 mod/addon!

HOW TO PLAY
1- Download ECWolf. You can get it here: http://maniacsvault.net/ecwolf/
2- Copy and paste all the .WL6 files (And the music.pk3) in this ZIP file in your ECWolf folder.
3- Each time you launch ECWolf drag music.pk3 on the executable (This contains
the music and PAR times)

WHAT THIS ZIP CONTAINS
Game Files: Contains the .WL6 and the music.pk3 files.
Story and Info: Contains a text file with the story, and another with info
about the maps (Such as author and name)
