Revision history

Version 0.3.12 - 02/20/2017

  • Fixed a bug in the map handler which prevented reading map planes at offset 0
  • Fixed some memory leaks in the dialog which provides statistics for individual map objects

Version 0.3.11 - 07/02/2016

  • Added a dialog which provides statistics for individual map objects
  • Fixed a typo in the clipboard manager dialog
  • Revised and updated help file

Version 0.3.10 - 06/15/2016

  • Fixed some bugs in the map compression algorithms
  • Fixed some tooltips not showing up over the buttons
  • Fixed some memory leaks in the WAD file handler
  • Added error handling code in case a project directory does not exist anymore

Version 0.3.9 - 06/14/2016

  • Fixed a bug in the map exporter dialog where closing the dialog with the window close button would export one or more maps into the previously chosen file format at the previously selected file location
  • Fixed a bug in the map rotation feature
  • Fixed grammar mistakes in several dialogs
  • Code optimizations in various parts of the editor
  • Reimplemented map compression algorithms

Version 0.3.8 - 02/12/2016

  • Fixed a bug where the "ZoomFactor" option was reset in the map definition file of a project when the project was reloaded

Version 0.3.7 - 10/23/2015

  • Support for Super 3D Noah's Ark
  • Changed color of the map grid
  • The validation checks for map filenames and map definition filenames are now case insensitive
  • Added ability to browse the list entries of the map symbols and map symbol types dialog with the arrow keys
  • Fixed some UI bugs in the map symbols and map symbol types dialog
  • Fixed a memory leak in the map symbol types dialog
  • Fixed a bug that prevented the "Check for update at startup" option from being written to the main configuration file
  • Revised and updated help file

Version 0.3.6 - 08/18/2015

  • Reimplemented flood fill functionality
  • Fixed a bug in the zoom slider control that didn't allow to directly select intermediary zoom factors
  • Fixed a bug in the solid rectangle tool that didn't allow to place rectangles that are only 1 tile wide

Version 0.3.5 - 07/15/2015

  • Autofloor functionality now takes secret rooms into consideration
  • Fixed grammar mistakes in several dialogs

Version 0.3.4 - 03/27/2015

  • Added WAD file support (Used in source ports such as ECWolf)
  • Added a project files selection dialog
  • Last opened project is now remembered by the "Open Project" option
  • The current map number is now displayed in the window title
  • Reduced the maximum map size to 180x180 tiles due to restrictions of the GAMEMAPS file format
  • Fixed several bugs in the "Find & Replace" functionality
  • Fixed broken Blake Stone support

Version 0.3.3 - 03/02/2013

  • Fixed a bug that prevented the map editing area from refreshing correctly when the "Show Floor" option was deactivated
  • Fixed a bug that prevented the software update checker from comparing program versions correctly
  • Fixed a bug that prevented the "Find & Replace" functionality from working correctly in all cases

Version 0.3.2 - 08/22/2011

  • Ability to display map information (Amount of map type objects in the currently selected map)
  • Ability to create map definition structures for unknown file extensions within HWE
  • The main window of HWE is now completely variable in size
  • Native support for the floppy disk version of Corridor 7
  • Ability to save maps in WDC compatible format
  • Additional exception handling
  • Revised and updated help file
  • Map clearing functionality
  • Numerous bug fixes

Version 0.3.1 - 06/05/2011

  • Additional confirmation dialog boxes
  • Map editing area is now variable in size
  • Zoom functionality for the map editing area
  • Support for the Spear of Destiny mission packs
  • Mouse wheel support for the vertical scrollbar of the map editing area
  • Ability to apply a new map size to all maps at once
  • Ability to search tiles / objects also by name
  • Ability to flip or rotate a copied map part
  • A MRU list (recent-files list)
  • Software update checker
  • Numerous bug fixes

Version 0.3 - 04/24/2011 (First public release)

  • Support for extended map names (32 characters)
  • Increased map size limit (from 196x196 tiles to 256x256 tiles)
  • Ability to easily search for and replace tiles / objects in a map
  • Revised and updated map definition file format (e.g. added map symbol categories and a value for the direction the map symbols are facing)
  • Additional map tools (e.g. flipping planes or rotating maps)
  • Revised and updated help file
  • Numerous bug fixes

Version 0.2 - 12/20/2010

  • An editor to add and edit the map objects inside of HWE
  • Support for Blake Stone: Aliens of Gold, Blake Stone: Planet Strike, Corridor 7 and Operation Body Count
  • Slightly improved Autofloor functionality (using a non-recursive algorithm now and is no longer bound to fixed floor codes)
  • Slightly improved map definition file format (comes closer to the MapEdit format now)
  • Slightly improved flood fill functionality (using a non-recursive algorithm now)
  • Slightly improved undo / redo functionality (using less memory now)
  • The first version of the help file
  • Numerous bug fixes

Version 0.1 - 04/20/2010 (First alpha release)

  • Map editing
  • Clipboarding
  • Exporting and importing (multiple) maps (in ChaosEdit, HWE, FloEdit, MapEdit, and WDC map format)
  • Unlimited map plane adding support
  • Unlimited map adding support
  • Undo / redo functionality
  • Autofloor functionality
  • High and low byte editing (e.g. useful for the textured floor and ceiling feature)
