Map configurations

Compression tags

  • RLEW-Tag - Set the RLEW-Tag for the maps.
  • NEARTAG - Set the NEARTAG for the maps.
  • FARTAG - Set the FARTAG for the maps.

You can restore the standard values by clicking on the "Set Defaults"-button.

Map compression

  • RLEW - Use RLEW compression for the maps only.
  • Carmack - Use RLEW and Carmack compression for the maps.

Map compression

  • Use 32 character map names - Here you can decide if your map names are either 16 or 32 characters long. (Note: This option applies to all maps and not individual ones!)
  • Save maps WDC compatible - Here you can decide if your maps should be saved in a WDC or id Software compatible format. (Note: This option is automatically activated when the "Save maps into WAD file" option is enabled.)
  • Save maps into WAD file - Here you can decide if your maps should be saved into a WAD file. (Note: This option is only useful for source ports with WAD file support such as ECWolf!)

Note: These values and options should only be changed, if you're going to modify the game engine or using a source port (e.g. ECWolf)!
