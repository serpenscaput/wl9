Map editing tools

HWE currently has the following map editing tools which should be familiar to you from programs like MS Paint:

Pencil tool Replace the selected tile with the currently selected tile.
Pick tile tool Select the tile in the currently selected tile.
Solid rectangle tool Draw a filled rectangle using the currently selected tile.
Tile selection tool Select a map section to copy or cut and paste the map section in a different or same map.
Flood fill tool Flood the map with the selected tile starting from the currently selected tile.

Additional tools can be found in the map commands toolbar and map menu.
