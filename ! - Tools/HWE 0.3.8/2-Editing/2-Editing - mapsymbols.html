Map symbols manager

Select plane

If you want to switch to a different plane simply click on the drop-down list on the left side in the map symbols manager and select the plane whose map objects you want to edit.

Select map symbols

You can change the order of the tiles in this list. This functionality was added so that you can group similar tiles together even when their map IDs are far apart. You can change the order by selecting the tile you want to move and using the arrow buttons to the right of the list.

Edit map symbol information

  • Name - The name for the map object.
  • Direction - The direction the map object is facing.
  • Type - The type of the map object. This list changes according to which plane the map object is in.

You can also manage the map symbol types by clicking on the "Manage map symbol types"-button.

Colors

This area provides you a 16-color palette which you can use to draw a symbol for your map object. You are allowed to select a primary color with the left mouse button and a secondary color with the right mouse button.

Symbol

This area is used to draw a map symbol for your map object. The map symbol will be displayed as icon in the tile selection list of the map editor. You can click a cell to draw it or you can press and hold the mouse button while moving the mouse in order to draw lines and other shapes manually.
