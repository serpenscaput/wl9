Editing :: Maps

To the right of the map area you will see map information, including the map name, number of guards, doors, etc.
Clicking the map button will bring up the list of wall tiles.
Clicking the tree button will bring up the list of object tile.
Clicking the wall button will toggle between the 'tile information' and 'tile graphics' views.
You can change the map's size by using the up/down arrows below the map's name.

WDC currently has the following edit tools, which should be familiar to you from MS Paint:
Pencil Replace the selected cell with the selected wall or object tile.
Flood Fill Flood the map with the selected tile starting from the selected cell.
Select Allows you to select a map section to copy or cut and lets you paste map sections.
Solid Rectangle Draw a filled rectangle using the selected tile.
Outline Rectangle Draw an outline rectangle using the left button tile for the outside and the right button tile for the inside.
Pick Tile Select the tile in the selected cell.
Manual Entry Allows you to manual enter a tile's numeric value.

Additional tools can be found in the Map Tools and Map Options menus.
