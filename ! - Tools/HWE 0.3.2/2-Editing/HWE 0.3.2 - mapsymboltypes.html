Map symbol types manager

Select map symbol type

In the list you can change the order of the map symbol types. You can change the order by selecting the type you want to move and using the buttons to the right of the list. This list is also used to select and edit already existing map symbol types.

Edit map symbol type information

  • Name - The name for the map symbol type.
  • ID - The ID for the map symbol type.

Note: The ID must be consist of a single character or several characters (written as one word) without blanks / spaces!
