Revision history

Version 0.3.2 - 08/22/2011

  • Ability to display map information (Amount of map type objects in the currently selected map)
  • Ability to create map definition structures for unknown file extensions within HWE
  • The main window of HWE is now completely variable in size
  • Native support for the floppy disk version of Corridor 7
  • Ability to save maps in WDC compatible format
  • Additional exception handling
  • Revised and updated help file
  • Map clearing functionality
  • Numerous bug fixes

Version 0.3.1 - 06/05/2011

  • Additional confirmation dialog boxes
  • Map editing area is now variable in size
  • Zoom functionality for the map editing area
  • Support for the Spear of Destiny mission packs
  • Mouse wheel support for the vertical scrollbar of the map editing area
  • Ability to apply a new map size to all maps at once
  • Ability to search tiles / objects also by name
  • Ability to flip or rotate a copied map part
  • A MRU list (recent-files list)
  • Software update checker
  • Numerous bug fixes

Version 0.3 - 04/24/2011 (First public release)

  • Support for extended map names (32 characters)
  • Increased map size limit (from 196x196 tiles to 256x256 tiles)
  • Ability to easily search for and replace tiles / objects in a map
  • Revised and updated map definition file format (e.g. added map symbol categories and a value for the direction the map symbols are facing)
  • Additional map tools (e.g. flipping planes or rotating maps)
  • Revised and updated help file
  • Numerous bug fixes

Version 0.2 - 12/20/2010

  • An editor to add and edit the map objects inside of HWE
  • Support for Blake Stone: Aliens of Gold, Blake Stone: Planet Strike, Corridor 7 and Operation Body Count
  • Slightly improved Autofloor functionality (using a non-recursive algorithm now and is no longer bound to fixed floor codes)
  • Slightly improved map definition file format (comes closer to the MapEdit format now)
  • Slightly improved flood fill functionality (using a non-recursive algorithm now)
  • Slightly improved undo / redo functionality (using less memory now)
  • The first version of the help file
  • Numerous bug fixes

Version 0.1 - 04/20/2010 (First alpha release)

  • Map editing
  • Clipboarding
  • Exporting and importing (multiple) maps (in ChaosEdit, HWE, FloEdit, MapEdit, and WDC map format)
  • Unlimited map plane adding support
  • Unlimited map adding support
  • Undo / redo functionality
  • Autofloor functionality
  • High and low byte editing (e.g. useful for the textured floor and ceiling feature)
