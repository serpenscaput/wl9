Open a project

Your first step to edit maps from a Wolfenstein 3D based game is to open the project folder which contains the maps. Select File | Open Project or press Ctrl+O or click on in the main window.

After you clicked on one of these options, the Select Directory dialog box will appear. Here you have to select the project folder of your Wolfenstein 3D based game.

If the selected maps have a file extension that doesn't exist as map definition in HWE, you'll see a dialog which asks you if you want to create a new map definition structure for these maps. Further information to this particular topic can be found in the tutorial "Creating map definitions for HWE".
