Tutorials :: Adding extended map names

If you are displaying the map name during the "Get Psyched" screen, 16 characters might not be enough. Checking "Extended map names" on the project information will convert the GAMEMAPS file so that it uses 32 characters instead. However, you need to make a few map changes to support the longer name
  1. Open ID_CA.H and change "name[16]" to "name[32]" in the maptype struct.
  2. Now go to ID_CA.C and change the mapfiletype struct to this: 
    typedef struct
{
	unsigned	RLEWtag;
	int		numplanes;		// added
	int		mapnamesize;		// added
	long		headeroffsets[100];
	byte		tileinfo[];
} mapfiletype;
And you're done with the code changes.
