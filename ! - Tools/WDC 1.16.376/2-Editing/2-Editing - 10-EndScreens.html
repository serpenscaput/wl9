Editing :: End Screens


The end screen editor tries to copy a simple text editor as closely as possible. The follow is a list of keys and their actions:
Letters, numbers, symbols Letters, numbers, symbols
[Space] [Space]
[Del] Clears the current cell
[Bakspace] Clears the current cell and goes to the previous cell
[Home] Go to the first cell for the current line
[End] Go to the last cell for the current line
NumberPad:1-9,-,+ Draws single width lines
Ctrl+NumberPad:1-9,-,+ Draws double width lines
Arrow keys Maneuver the screen
Ctrl+A..Z ASCII characters 1 through 26
Ctrl+1..8, Ctrl+Shift+1..8 Change the foreground color
Alt+1..8, Alt+Shift+1..8 Change the background color
Alt+NumberPad1..3 Shaded blocks

The rest should be pretty self-evident.
