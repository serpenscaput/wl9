Error Message Guide :: Critical Errors

Error Description Resolution
Error 1601 in PreCache: Error pre-caching picture (VGAGRAPH #n) Turning image pre-caching off will cause this error to go away if you are receiving it. You might let me know about it if it seems to be more than a memory issue.
Error 1602 in PreCache: Error pre-caching wall (VSWAP #n) Turning image pre-caching off will cause this error to go away if you are receiving it. You might let me know about it if it seems to be more than a memory issue.
Error 1603 in PreCache: Error pre-caching sprite (VSWAP #n) Turning image pre-caching off will cause this error to go away if you are receiving it. You might let me know about it if it seems to be more than a memory issue.
Error 1604 in CatBuster: Could not extract UNP.EXE! UNP.EXE is needed in order to extract some data files from game EXEs for Corridor 7 and some other games. If UNP can't be extracted, CatBuster can't get extract the files from the EXE. Let me know if this happens to you.
Error 1605 in BuildNewVGAGRAPHOrder: Unknown chunk type: [datatype] WDC couldn't determine what chunk type a certain VGAGRAPH chunk was. Let me know about this first. I will ask you to send your VGAGRAPH, VGAHEAD, and VGADICT files (in a ZIP file) if I need to look at them, but please wait for me to ask first.
Error 1606 in frmListChunks_Load: Unknown chunk type: [datatype] WDC couldn't determine what chunk type a certain VGAGRAPH chunk was. Let me know about this first. I will ask you to send your VGAGRAPH, VGAHEAD, and VGADICT files (in a ZIP file) if I need to look at them, but please wait for me to ask first.
Error 1607 in CatBuster: EXE decompression failed for [filename] UNP.EXE was not able to decompress your EXE to the correct size. You probably just have a different version of the file than I used to set up the CatBuster.ini. If you're handy with a hex editor, you can set up CatBuster.ini with the correct information.
Error 1608 in clsVGAGraph.LoadWolfFonts: Error loading font #n This is pretty serious... let me know about it.
Error 1609 in clsGameMaps.ExportMultiMap: Error loading map data for file export This is pretty serious... let me know about it.
Error 1610 in clsVSWAP.SaveVSwap: Error reloading original VSWAP. Please shutdown and restart the program. This is pretty serious... let me know about it.
Various other critical errors Be sure you have the latest version and try it again. If you still get the error, let me know about it. Error.log will have an entry for all critical errors. Please include this entry in your email.
