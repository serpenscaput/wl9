General :: MEDC

MEDC stands for "MapEdit Data Convertor" and it is an easy-to-use program that converts MapEdit's MAPDATA and OBJDATA files into WDC's Map Data file format.

To convert files, simply select the MAPDATA and OBJDATA files and click 'Convert'.

If you want to change how some of the symbols will be converted, simply select the symbol using the two lists and use the 7x7 grid to set the symbol.

Left clicking a grid cell will alternate between the primary color and no color.
Right clicking a grid cell will alternate between the secondary color and no color.
