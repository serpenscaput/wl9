Wolfenstein 3D - Red Sunrise SDL
================================

Overview
--------

This is an enhanced and refined version of the "Red Sunrise" map set
from BAKTRA Software. It is powered by the Wolf4SDL engine. Red Sunrise
was originally released in 2005. The SDL version is from 2022.

The map set is using original graphics and sounds, it is suited for
players who like the original game.

Features
--------
Maps and Graphics
- 60 Original Maps
- Some subtle VGAGRAPH changes (intro screen, main menu)
Engine
- Episode names
- Par times for maps
- Demo playback disabled

Differences from the original version
-------------------------------------
All maps were subtly refined to implement better mapping and object
placement practices. The very nature of the map set is strongly preserved, though.
