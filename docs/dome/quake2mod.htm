Quake II/Wolfenstein 3D

 

This add-on shows what Wolfenstein 3D could look like with a more modern engine.

All shareware levels (episode 1 - levels 1 thru 10) are included. They were made by AklashPahk, from Brazil. Also included is a Doom 2 level (map 32) by Arno Ansems.

Only the walls are shown in this set. The objects and actors have not been added yet.

 

Instructions:

  • Download Wolfen.zip (4.17Mb)
  • Unzip, and place the pak5.pak file into the \quake2\baseq2 folder
  • Run Quake II, and from the menu press the key to the left of the '1' key (to bring up the console)
  • Type map we1m1 for the first level (for other levels, type map we1m2 etc, and for the Doom level type map d2map32)

Quake II is owned by id software and distributed by Activision. You can get it from The CD-Rom Shop.
